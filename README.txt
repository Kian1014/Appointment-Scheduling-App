Customer and Appointment Record View - Allows CRUD functionality through a GUI of 
customers and appointments as well as report generation.

Author: Kian Haghnegahdar
Contact Info: kianhgh93@gmail.com
Application Version: 1.01
Date: 3/15/2021
IDE: Intellij IDEA 2020.3.2
JDK: 11.0.9
JAVA FX ver: 11.0.2	
SQL Driver: 8.0.23

Directions: After running program use main screen to navigate between reports screen, appointments screen, and customers screen. Appointment and customer screens have buttons for creation, updating, and deleting respective objects. Reports page uses combo box selection for criteria and button to generate reports.

Additional report: Displays number of customers in a given division.
